# Changelog
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.4.1]
### Changed
- Allow SumatraPDF as much time as needed to complete the print process (was previously limited to 30 seconds total execution time, which cut off larger print jobs)

## [1.4.0]
### Changed
- Print Manager now honors the copies parameter

## [1.3.1]
### Changed
- Print Manager can handle printers and file paths containing spaces

## [1.3.0] [r22.08]
### Added
- First release!
